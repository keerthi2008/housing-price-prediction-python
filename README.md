# housing-price-prediction-python

This project focuses on predicting housing prices using machine learning techniques. It involves data preprocessing, exploratory data analysis (EDA), model building, and prediction.

🎯 Objectives

Clean and preprocess housing dataset

Perform exploratory data analysis (EDA)

Build a predictive model using regression

Evaluate model performance

Predict house prices based on input features

📊 Dataset Features

Area (sqft)

Number of bedrooms

Number of bathrooms

Location

Parking availability

Price (Target Variable)

⚙️ Technologies Used

Python 🐍

Pandas

NumPy

Matplotlib & Seaborn

Scikit-learn

🔄 Project Workflow

1️⃣ Data Collection

2️⃣ Data Cleaning

3️⃣ Exploratory Data Analysis (EDA)

4️⃣ Feature Engineering

5️⃣ Model Training (Linear Regression)

6️⃣ Model Evaluation

7️⃣ Prediction

📈 Model Used

Linear Regression
